# AR-handbutton

AR-handbutton is a virtual reality (VR) hand button interaction demo built using Three.js and the WebXR API.

## Demo
You can view the demo at https://code4fukui.github.io/ar-handbutton/

## Features
- Allows users to press a virtual button using their hand in AR/VR
- Provides haptic feedback and sound effects when the button is pressed
- Includes a rotating cube as an example of an interactive 3D object

## Requirements
- A WebXR-enabled web browser (e.g., Chrome, Firefox) on a device with AR/VR capabilities

## Usage
To run the project locally:

1. Clone the repository:
   ```
   git clone https://github.com/code4fukui/ar-handbutton.git
   ```
2. Open the `index.html` file in a web browser.
